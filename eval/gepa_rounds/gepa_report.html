CIPHER — GEPA Calibration Report

Goal: find a captcha generator configuration at which Gemini Flash achieves ~3% EM (the sweet spot between "100% trivial" and "0% hallucination"). Fitness function: -|EM - 0.03| - 0.5·(missing/n).

TL;DR: four rounds, 5 programs each (R4 used 8 samples/program instead of 4). R3 prog_3 reaches EM=0% with the key length predicted correctly (a near-miss), making it the strongest candidate for R5. R4 is invalid: Gemini API was down, Claude was substituted, and Claude hallucinates on captchas (char-acc 2–11%, which matches our prior finding in project memory).
